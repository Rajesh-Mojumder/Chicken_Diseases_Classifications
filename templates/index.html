About This Project

The Chicken Disease Classification System is an AI-powered application that detects poultry diseases using image analysis. It applies deep learning (VGG16) to classify whether a chicken is Healthy or infected with Coccidiosis.

This system enables quick and reliable predictions, helping reduce manual inspection efforts and supporting early disease detection. It demonstrates a complete ML pipeline from data processing to deployment.

  • Automated disease detection
  • User-friendly interface
  • Early diagnosis support
  • Real-world AI implementation
